# QBook

QBook 是一个跨平台记账应用，目标覆盖一木记账的核心能力，并基于 Flutter 建立 Android、iOS、Web、Desktop 的统一代码库。

## 当前状态

- 已完成产品 PRD 和 Flutter 架构文档
- 已完成 OpenSpec proposal / design / specs / tasks
- 已建立第一版 Flutter 项目骨架

## 目录

- `docs/`：产品和架构文档
- `openspec/`：变更驱动规范与任务
- `lib/`：Flutter 客户端代码

## 注意

当前环境未安装 `flutter` / `dart` 工具链，因此原生 runner、依赖拉取和 Drift 代码生成尚未执行。
