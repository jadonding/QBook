## 新增需求

### 需求：系统必须提供首页财务概览
系统必须在首页展示当前账本的关键财务数据，使用户能快速了解近期收支和资产状态。

#### 场景：显示本月概览
- **当** 用户进入首页
- **那么** 系统必须展示本月收入、本月支出、本月结余和预算进度

#### 场景：显示最近账单
- **当** 用户进入首页
- **那么** 系统必须展示最近账单列表并允许跳转到账单详情

### 需求：系统必须提供多维度统计分析
系统必须支持按日、周、月、年查看收支、分类占比、趋势和预算执行情况。

#### 场景：查看月度统计
- **当** 用户切换到统计页并选择当前月份
- **那么** 系统必须展示该月份的收入、支出、结余和分类占比

#### 场景：查看年度趋势
- **当** 用户在统计页选择年度视图
- **那么** 系统必须展示按月份汇总的年度收支趋势

#### 场景：查看预算执行
- **当** 用户查看某个预算周期的统计信息
- **那么** 系统必须展示预算总额、已用金额和剩余额度
